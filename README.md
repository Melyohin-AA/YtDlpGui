# YtDlpGui

YtDlpGui provides simple and limited GUI for [yt-dlp](https://github.com/yt-dlp/yt-dlp).

This project is not affilated with the **yt-dlp** dev-team.

## Requirements

* Windows 10 x64
* .NET 8.0 runtime
* yt-dlp installed and added to **PATH**
* [Node.js](https://nodejs.org/) installed and added to **PATH** - required for running [EJS challenge solver scripts](https://github.com/yt-dlp/yt-dlp/wiki/EJS)
